Submit search results to the Virtual Collection Registry

Provide a name for the collection and click the button below to create a new Virtual Collection of the current search results (999 metadata records).
After submission you will be taken to the editor of the Virtual Collection where you will be able to further edit the newly created collection. You may be prompted to login to the Virtual Collection Registry first.
